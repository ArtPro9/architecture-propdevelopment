| Роль   | Права роли                                                                                 | Группы пользователей                                   |
| ------ | ------------------------------------------------------------------------------------------ | ------------------------------------------------------ |
| Viewer | Только просмотр ресурсов кластера                                                          | Менеджеры, бизнес-аналитики,  сотрудники отдела продаж |
| Dev    | Право на настройку кластера, но без доступа к прод секретам и критическим конфигурациям    | DevOps-инженеры, разработчики                          |
| Admin  | Полный доступ к кластеру, включая управление секретами, конфигурациями и другими ресурсами | Системные администраторы, технические руководители     |

## Настройка K8S

```shell
 kubectl apply -f roles.yml
 kubectl apply -f users.yml
 kubectl apply -f bindings.yml
```
